# Подсказки по командной строке

Листинг текущей директории
Windows:
```sh

```
Удаление файда
```sh
del <имя файла>
```
Посмотреть commit
```sh
log
```

Сокращенная версия log
```sh
log --oneline
```

После использования log пропадает ввод в терминале
```sh
Чтобы вернуть ввод нажми на <q>
```

Чтобы переключаться между сейвами
```sh
checkout
```
